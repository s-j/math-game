# Times Table Challenge

A colourful browser game that helps children practise multiplication tables with quick-fire multiple choice questions, instant feedback and optional countdown timers.

## Features

- Adjustable difficulty: choose the highest times table to include (from 5× to 12×).
- Optional thinking timer (5, 10 or 15 seconds) to add a little excitement.
- Multiple choice answers with streak, accuracy and total score tracking.
- Friendly hints that rotate to reinforce multiplication strategies.
- Works entirely offline—just open the HTML file in a browser.

## Getting started

1. Clone or download this repository.
2. Open `index.html` in any modern browser (Chrome, Edge, Firefox, Safari).
3. Adjust the sliders and dropdown to set the challenge, then start tapping the answers.

## Customising

- Tweak the colour palette by editing `style.css`.
- Update or add new strategy hints by modifying the `tips` array in `script.js`.
- Change the range of numbers that appear by editing the call to `randomInt` inside `presentQuestion()`.

Have fun mastering the times tables!
